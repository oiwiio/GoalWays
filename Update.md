# 🛠 Технологический стек проекта

**Стек** — это набор технологий, инструментов и библиотек, которые используются в разработке приложения.  
Можно представить его как **кухню ресторана**, где каждый инструмент и ингредиент выполняет свою роль для создания готового блюда (продукта).

##  Фронтенд-стек мобильного приложения

###  **Ядро**
```javascript
React Native + TypeScript
```
- **React Native** — фреймворк для кроссплатформенной разработки
- **TypeScript** — статическая типизация для надёжности кода

###  **Управление состоянием**
```javascript
Zustand
```
- Легковесное и простое решение для глобального состояния
- Минимальный бойлерплейт, интуитивный API

###  **Навигация**
```javascript
React Navigation
```
- Стандартная библиотека для навигации в React Native
- Стек-навигация, табы, дроверы

###  **Работа с сетью**
```javascript
Axios + TanStack Query (React Query)
```
- **Axios** — HTTP-клиент для запросов к API
- **TanStack Query** — кэширование, синхронизация и управление серверным состоянием

###  **Дополнительные технологии** (под вопросом)
```javascript
// Хранилище данных:
AsyncStorage & SQLite

// UI-библиотека:
React Native Paper
```
- **AsyncStorage** — простое key-value хранилище
- **SQLite** — локальная база данных для сложных данных
- **React Native Paper** — Material Design компоненты для React Native

##  **Преимущества выбранного стека:**

| Технология | Преимущества |
|------------|--------------|
| **React Native** | Кроссплатформенность, горячая перезагрузка, большое комьюнити |
| **TypeScript** | Безопасность типов, автодополнение, раннее обнаружение ошибок |
| **Zustand** | Простота использования, минимальный код, отличная производительность |
